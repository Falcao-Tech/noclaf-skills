# Skills

Skills são **model-invoked**: o agente ativa sozinho quando o contexto bate com a `description` do `SKILL.md`. Ficam em `<seção>/<nome>/SKILL.md`.

## Seções

- **[[skills/dev/README|dev]]** — skills de engenharia (código, review, design), subdivididas por stack.
- **[[skills/productivity/README|productivity]]** — fluxo de trabalho e escrita, fora do código. (vazio)

> A seção é só organização — **não** muda como a skill é invocada; serve pra você e o time acharem a skill, não pro agente. Regra de idioma da vault: skill/command/doc em **pt-BR**, código em **inglês**.
