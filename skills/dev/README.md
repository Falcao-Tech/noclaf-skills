# Skills · dev

Pra escrever, revisar e projetar código. Agrupadas por **stack** — a pasta é organização, não muda a invocação.

## general — agnóstico de stack

- [[caveman]] — resposta ultra-comprimida (~75% menos tokens).
- [[council]] — conselho de 5 lentes que mata concordância fácil numa decisão.
- [[drytify]] — acha e remove duplicação real de código (DRY com bom senso).
- [[grill-me]] — te sabatina sobre um plano/design até resolver cada decisão.
- [[handoff]] — compacta a conversa num doc de continuidade pra outro agente.
- [[review-changes]] — review de qualidade/limpeza do diff (não caça bug).
- [[zoom-out]] — sobe um nível de abstração e mapeia módulos/callers.

## front-end

- [[animation-vocabulary]] — glossário reverso: descrição vaga → termo exato de motion.
- [[emil-design-eng]] — filosofia de polish de UI e craft (Emil Kowalski).
- [[review-animations]] — revisa código de animation contra uma régua alta de craft.
